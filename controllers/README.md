# Controller Library

## Installation

```sh
yarn add @hopsoft/controllers
```

## Usage

```js
import ExampleController from '@hopsoft/controllers'
```

## Contributing

Standardize your code before pushing to GitHub.

```sh
yarn prettier-standard:format
```

## Publishing

```sh
yarn build
yarn publish
```
